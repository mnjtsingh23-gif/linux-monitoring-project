# Linux Monitoring & Backup Automation

## 📌 Project Overview
This project is a Bash script that monitors system performance and creates automated backups.

## 🚀 Features
- CPU, Memory, Disk monitoring
- Top processes analysis
- Automated backup using tar
- Cron job scheduling

## 🛠 Tools Used
- Linux (Ubuntu/CentOS)
- Bash Scripting
- Cron Jobs

## ▶️ How to Run
chmod +x monitor.sh  
./monitor.sh

## 📂 Output
- System report in terminal
- Backup file (.tar.gz)
